Como obtener los datos de Chrome y llenar los campos

1. Abre Chrome en el perfil que ya tiene sesion de TikTok.

2. Escribe chrome://version/ y busca la linea Profile Path.

3. Copia el Profile Path completo. Ejemplo: C:/Users/TuUsuario/AppData/Local/Google/Chrome/User Data/Profile 1

4. Ruta User Data Chrome: pega solo hasta User Data. Ejemplo: C:/Users/TuUsuario/AppData/Local/Google/Chrome/User Data

5. Perfil de Chrome: pega solo el nombre final del perfil. Ejemplo: Profile 1 o Default

6. Nombre: alias de la cuenta para identificarla en el panel.

7. Velocidad: 1.0, 1.1 o 1.2. Estado inicial: Activa si quieres que suba ahora.

8. Tipo de cuenta: Principal o Secundaria segun tu estrategia de plantillas.
